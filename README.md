# Spotify UI Clone

Replica da interface do Spotify focada em fidelidade visual, componentização e experiência do usuário.

![Preview](./docs/spotify_original_demo.gif)

## Clone

![Preview](./docs/spotify_ui_clone_demo.gif)

## Tecnologias

- React
- SCSS (Sass)
- Swiper.js

## Funcionalidades

- Interface inspirada no Spotify
- Carrosséis interativos
- Sidebar fixa
- Scroll customizado
- Text overflow com ellipsis
- Layout organizado por seções (músicas, artistas, destaques)

## Aprendizados

- Estruturação de layouts complexos com Flexbox e Grid
- Reutilização de componentes
- Organização de estilos com SCSS modular
- Integração de bibliotecas externas
- Controle de overflow e truncamento de texto
